# Types module

Shared TypeScript types for wills, roles, and form data.

## Contents

- **`index.ts`** — `Will`, `WillInsert`, `WillUpdate`, `WillStatus`, `WalletRole`, `WillWithRole`.

## Usage

```ts
import type { Will, WillWithRole, WalletRole } from "@/lib/modules/types";
```

Used across app, API routes, and other modules.
